Two different clocks: the swing timer vs the GCD

WoW melee runs on two separate timers, and mixing them up is the #1 beginner confusion. Let's name them now.

Swing timer

How often your white auto-attack fires. It equals your weapon's Speed — a slow axe might be 3.0s, a quick dagger 1.5s. This is the timer you already have.

Global cooldown (GCD)

A fixed 1.5s lockout after you use an ability (like a spell or a special strike) before you can use another. It does not touch your auto-attack.

Level

A number on every fighter. The difference between attacker and defender level drives miss/dodge/crit — the heart of the next lessons.

◈ why they're separate

Your weapon keeps swinging on its own rhythm no matter what abilities you press — that's the swing timer. The GCD only stops you from chaining abilities too fast. (And Heroic Strike, your future signature move, is sneaky: it sits off the GCD and rides your next swing. More on that in Lesson 32.)

Part A — swing at your weapon's speed

Right now your SwingTimer has a fixed wait time set in the editor. Let's drive it from a weapon speed variable instead, so it's a real stat you can tune (and later, swap per weapon).

GDScript · player.gd
# with the other @export vars near the top:
@export var weapon_speed = 2.0    # seconds per swing (Worn Shortsword is ~1.9 in Classic)

# in _ready(), set the timer from it:
func _ready():
    add_to_group("player")
    swing_timer.wait_time = weapon_speed     # the swing timer IS the weapon speed
    health_bar.max_value = max_health
    health_bar.value = health
@export var weapon_speed = 2.0 — your weapon's Speed stat. In WoW the white swing timer simply is this number (no haste). Source: wow-combat-values.md §1.
swing_timer.wait_time = weapon_speed — feed it into the Timer when the game starts. Change weapon_speed in the Inspector and your attack cadence changes with it.
▶

Press F5, target a wolf, attack. Try weapon_speed = 3.5 (ponderous) then 1.2 (frantic) in the Inspector to feel the difference. Set it back to 2.0.

Part B — the GCD (concept now, code later)

There's nothing to wire for the GCD yet — it only matters once you have an ability to gate, and your first one (Heroic Strike) is Lesson 32. We're naming it now so the idea is in place:

▲ don't put your auto-attack on the GCD

A common mistake is making every attack respect a 1.5s GCD. That would break fast weapons. White swings ride the swing timer (weapon speed); only abilities ride the GCD. We'll add a tiny GCD timer when the first ability needs it — not before (no dead code).

Part C — give everyone a level

A level is just an exported number. Add it to the Player and the Wolf, and show the wolf's level above its head so the danger is readable at a glance.

Player

GDScript · player.gd
# with the other @export vars:
@export var level = 1

Wolf — add a level label to the scene

1

Open wolf.tscn. Select the Wolf root → + Add Child Node → Label. Rename it LevelLabel.

2

In the Inspector set its Position to about (−10, −42) (just above the health bar). Shrink the font if you like (Theme Overrides → Font Sizes → Font Size ≈ 10).

3

Ctrl+S to save wolf.tscn.

GDScript · wolf.gd
# with the other vars / @onready vars near the top:
@export var level = 1
@onready var level_label = $LevelLabel

# in _ready(), show it:
func _ready():
    spawn = global_position
    health_bar.max_value = max_health
    health_bar.value = health
    level_label.text = "L" + str(level)
@export var level = 1 on the wolf — because it's exported, each wolf instance can have its own level set in the Inspector, with no separate script. This is the payoff of @export from L10.
level_label.text = "L" + str(level) — show it. str() turns the number into text (same trick as the damage numbers).

Set each wolf's level

1

Back in main.tscn, select Wolf → in the Inspector find Level (your exported var) → leave it 1.

2

Select Wolf2 → set Level = 2. Select Wolf3 → set Level = 3. Same scene, three different levels.

▶

Press F5. Each wolf wears its level — L1, L2, L3 — above its bar. You're L1 facing a pack that's mostly tougher than you. Next lesson, that gap starts to bite. 📊
