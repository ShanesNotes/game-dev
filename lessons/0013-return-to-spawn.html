Reset: Return to Spawn

Leave the wolf's aura and it walks back to exactly where it started, then waits there to be pulled again.

The idea

Right now, when you leave the aura, target becomes null and the wolf just stops wherever it happens to be. We want it to remember its spawn and walk back. Two small additions:

remember home

In _ready() (runs once at start), store the wolf's starting position in a variable, spawn.

walk back when idle

When there's no target, head toward spawn until you're basically on it, then stop.

It's the same "move toward a point" math you use for chasing — just pointed at spawn instead of the player.

The code

Only the top of the file and _physics_process change. Replace wolf.gd with this:

GDScript · wolf.gd
extends CharacterBody2D

@export var speed = 120
@export var melee_range = 40.0
@export var swing_damage = 1

@onready var sprite = $Sprite2D
@onready var swing_timer = $SwingTimer

var target = null
var spawn = Vector2.ZERO          # where the wolf started

func _ready():
    spawn = global_position          # remember home, once

func _physics_process(delta):
    if target != null:
        # in combat: close to melee, then hold
        if global_position.distance_to(target.global_position) > melee_range:
            velocity = (target.global_position - global_position).normalized() * speed
        else:
            velocity = Vector2.ZERO
    else:
        # no target: walk back to spawn
        if global_position.distance_to(spawn) > 4.0:
            velocity = (spawn - global_position).normalized() * speed
        else:
            velocity = Vector2.ZERO
    move_and_slide()
    face_target()

func _on_aggro_area_body_entered(body):
    if body.name == "Player":
        target = body
        swing_timer.start()

func _on_aggro_area_body_exited(body):
    if body.name == "Player":
        target = null
        swing_timer.stop()

func _on_swing_timer_timeout():
    if target != null and global_position.distance_to(target.global_position) <= melee_range + 16:
        print("The wolf bites you for ", swing_damage, "!")

func face_target():
    var face_x = velocity.x
    if target != null:
        face_x = target.global_position.x - global_position.x
    if face_x != 0:
        sprite.flip_h = face_x < 0
func _ready(): spawn = global_position — captures home once. Because we fixed the wolf's transform last lesson, global_position is now truly where the wolf sits — so this records the right spot.
else: branch — no target → go home. Same normalize-and-move pattern as chasing, aimed at spawn. The > 4.0 check stops it jittering once it arrives.
▶

Press F5. Pull the wolf, lead it across the field, then step out of its aura — it turns and walks straight back to its spawn, then waits. Pull it again and it re-engages. 🎉
