# Game Dev — Godot Learning Workspace

A `teach`-skill workspace for learning Godot game development.

## How to use it

From this folder, run:

```
cd ~/game-dev
claude
/teach godot
```

The `teach` skill drives stateful, multi-session learning. It produces
**lessons** (one tightly-scoped HTML file each, opened in your browser) grounded
in your **mission**.

## What's here

| Path                | Purpose                                                        |
|---------------------|---------------------------------------------------------------|
| `MISSION.md`        | *Why* you're learning — grounds every lesson. **Fill first.** |
| `RESOURCES.md`      | Curated, trusted sources (knowledge + communities).           |
| `NOTES.md`          | Your teaching preferences + verified environment facts.       |
| `reference/`        | Compressed cheat-sheets/glossaries you revisit (HTML).        |
| `lessons/`          | Self-contained interactive lessons, `0001-...html`.           |
| `learning-records/` | What you've learned — drives what to teach next.              |
| `projects/`         | Your actual Godot projects.                                   |

## Environment
Godot **4.6.2 stable (mono)** at `~/.local/bin/godot`. See `NOTES.md`.
